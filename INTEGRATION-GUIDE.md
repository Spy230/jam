# Руководство по интеграции обновлений сайта

## Обзор изменений

Были созданы следующие файлы с новыми разделами и стилями:

1. **custom-styles.css** - Дополнительные стили для новых разделов
2. **custom-scripts.js** - JavaScript для интерактивности
3. **sections.html** - HTML-разметка новых разделов
4. **footer-contacts.html** - Обновленный футер с контактами

## Шаги интеграции

### 1. Добавьте ссылки в `<head>` секцию index.html

Уже добавлено:
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<link href="https://unpkg.com/aos@2.3.1/dist/aos.css" rel="stylesheet">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
```

Добавьте после существующих стилей:
```html
<link rel="stylesheet" href="custom-styles.css">
```

### 2. Замените или обновите разделы в index.html

#### Промо-акция
Вставьте после секции "О нас" (about-us):
```html
<!-- Содержимое из sections.html: ПРОМО-АКЦИЯ -->
```

#### Услуги
Замените существующий блок `.services` на:
```html
<!-- Содержимое из sections.html: ОБНОВЛЕННЫЙ РАЗДЕЛ УСЛУГ -->
```

#### Наши Мастера (новый раздел)
Вставьте после раздела услуг:
```html
<!-- Содержимое из sections.html: РАЗДЕЛ НАШИ МАСТЕРА -->
```

#### Интерьер
Замените существующий блок `.interior` на:
```html
<!-- Содержимое из sections.html: ОБНОВЛЕННЫЙ РАЗДЕЛ ИНТЕРЬЕР -->
```

#### Отзывы (новый раздел)
Вставьте после раздела интерьер:
```html
<!-- Содержимое из sections.html: РАЗДЕЛ ОТЗЫВЫ -->
```

#### Контакты и Футер
Замените существующие секции `.map-section` и `<footer>` на:
```html
<!-- Содержимое из footer-contacts.html -->
```

### 3. Добавьте модальное окно

Перед закрывающим тегом `</body>` добавьте:
```html
<!-- Содержимое из sections.html: МОДАЛЬНОЕ ОКНО -->
```

### 4. Подключите скрипты

Перед закрывающим тегом `</body>` добавьте:
```html
<!-- Bootstrap JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
<!-- AOS Animation -->
<script src="https://unpkg.com/aos@2.3.1/dist/aos.js"></script>
<!-- Custom Scripts -->
<script src="custom-scripts.js"></script>
```

## Новые функции

### 1. Промо-акция
- Анимированный баннер с эффектом pulse
- Привлекает внимание к специальному предложению

### 2. Обновленные услуги
- Grid-layout с адаптивным дизайном
- Анимация появления при скролле (AOS)
- Hover-эффекты на карточках

### 3. Раздел "Наши Мастера"
- Карусели с фотографиями мастеров
- Hover-эффекты на карточках
- Клик по фото открывает полноэкранный просмотр

### 4. Обновленный интерьер
- Автоматическая карусель
- Индикаторы и контроли навигации
- Клик по фото для полноэкранного просмотра
- Кнопка "Посмотреть весь интерьер"

### 5. Отзывы
- Карусель с отзывами клиентов
- Аватары, рейтинги, даты
- Автоматическая прокрутка

### 6. Контакты
- Полная контактная информация
- Интеграция с мессенджерами (WhatsApp, Telegram)
- Кнопка заказа такси Яндекс Go
- Встроенная карта Яндекс

### 7. Модальное окно
- Полноэкранный просмотр изображений
- Работает для всех каруселей
- Закрытие по клику на крестик или вне изображения

## Адаптивность

Все разделы полностью адаптивны:
- Desktop: 3-4 карточки в ряд
- Tablet: 2 карточки в ряд
- Mobile: 1 карточка в ряд

## Анимации

Используется библиотека AOS для плавных анимаций при скролле:
- `fade-up` - появление снизу
- `fade-right` - появление справа
- `fade-left` - появление слева
- `zoom-in` - увеличение

## Поддержка браузеров

- Chrome/Edge (последние версии)
- Firefox (последние версии)
- Safari (последние версии)
- Мобильные браузеры (iOS Safari, Chrome Mobile)
